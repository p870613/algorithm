### sol
- 使用 unordered_set 存
- traverse array
    - 先確認 nums[i]-1 有沒有存在
        - 有 不繼續檢查
        - 無 用迴圈往右邊檢查
        - example
            - [100,4,200,1,3,2] => unordered_set {1, 2, 3, 4, 100, 200}
            - traverse
                - 1 => 往右邊檢查, [1, 2, 3, 4] => len: 4
                - 2 => 左邊有值, 不檢查
                - 3 => 左邊有值, 不檢查
                - 4 => 左邊有值, 不檢查
                - 100 => 往右邊檢查, => len: 1
                - 200 => 往右邊檢查, => len: 1
